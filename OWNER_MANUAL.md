# Hoods Cleaning – Owner Manual

This guide explains how to use the system and create new service pages. It is written for non‑technical users.

---

## 1) Sign in
1. Go to: https://hoodscleaning.net/admin/login
2. Enter the admin password.
3. Click **Sign in**.

---

## 2) Create one page (City)
1. Go to **Service Pages**.
2. Click **Generate Draft**.
3. Select the **Keyword** (the service name).
4. Select **City**.
5. Type the city name.
6. (Optional) Type the county name.
7. Confirm the slug (the link). It should match the city.
8. Click **Generate Draft**.

---

## 3) Create one page (ZIP)
1. Go to **Service Pages**.
2. Click **Generate Draft**.
3. Select the **Keyword** (the service name).
4. Select **ZIP Code**.
5. Type the ZIP code.
6. (Optional) Type the county name.
7. Confirm the slug (the link).
8. Click **Generate Draft**.

---

## 4) Review drafts
1. Go to **Service Pages**.
2. Click **Recent Pages**.
3. Review each draft:
   - Read the content.
   - Check the flags (warnings).

---

## 5) Publish or delete
### Publish
1. Find the draft.
2. Click **Publish**.
3. If a warning appears, confirm only if you approve it.

### Delete
1. Find the draft.
2. Click **Delete**.

---

## 6) Create many pages at once (Bulk list)
1. Go to **Service Pages**.
2. Click **Bulk CSV**.
3. Copy this format:

```
primary_keyword_slug,city,zip,county,intent,slug
hood-cleaning,Los Angeles,,Los Angeles,information,
hood-cleaning,,91311,Los Angeles,information,/hood-cleaning-91311
```

4. Paste your list into the box.
5. Click **Generate Drafts**.
6. Watch the log at the bottom.

---

## 7) Check a page
1. Go to the **Locations** page.
2. Search for the city.
3. Click the city to open the page.

---

## 8) Tips
- Only publish pages you approve.
- Use real city names and correct ZIPs.
- Keep the phone and address consistent.
- If a draft looks wrong, delete it and generate again.

---

## 9) Common questions
**Q: Why can’t I see my page?**
- The page is still a draft. Publish it first.

**Q: A draft has warnings. Should I publish?**
- Only publish if the content still looks correct.

**Q: How do I change the phone or address?**
- Ask the admin to update the global settings.

---

## 10) Support
If you need help, contact the system admin.
